# EDA (Exploratory Data Analysis)
- 탐색적 데이터 분석
- 수집한 데이터를 다양한 각도로 관찰하고 이해하는 과정

## 특성 선택하기 (feature selection)
- 원한는 변수들을 선택
  - 성별
  - 태어난 연도
  - 혼인 상태
  - 종교
  - 월급
  - 직업코드
  - 지역코드


## 1. 성별에 따른 월급 차이

1. 변수 검토
  - 성별
  - 월급
2. 변수 간 관계 분석
  - 성별 - 월급 평균표
  - 그래프 그리기

    - 나이
    - 월급

    - 나이에 따른 월급 평균표
    - 그래프 그리기

1. 변수 검토
    - 연령대
    - 성별
    - 월급
2. 변수 간 관계 분석

    - 그래프
